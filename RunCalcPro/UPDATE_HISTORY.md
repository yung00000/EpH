# RunCals Pro - Update History

## Version 1.2.1 (January 10, 2026)

### New Features
- **(New) Race Events**: Track your upcoming races with countdown timer and event management
- **(New) Running Tips**: Access running articles with offline caching and pull-to-refresh
- **(New) Swipe-to-delete**: Easily manage events and history items with swipe gestures
- **(New) Enhanced UI**: Improved navigation with bottom tabs and modern modal pickers

### Improvements
- Better theme synchronization between system and app settings
- Improved date picker for race events
- Enhanced settings modal with blue accent colors for active elements

### Bug Fixes
- Fixed theme state synchronization issues
- Improved language switcher consistency
- Better handling of automatic theme mode

## Version 1.2.0 (January 1, 2026)

### New Features
- Multi-language support (English and Traditional Chinese)
- Theme system with automatic, light, and dark modes
- Calculation history for both EpH and Pacing calculators
- Offline storage for user preferences and history

### Improvements
- Modern UI redesign with better color scheme
- Improved input validation and error handling
- Enhanced calculator result display

### Bug Fixes
- Fixed time format parsing issues
- Improved pace calculation accuracy
- Better error messages for invalid inputs
